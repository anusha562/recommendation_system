# Movie Recommendation System

## Overview
This project implements a movie recommendation system that suggests movies based on user preferences and historical ratings. The system uses machine learning algorithms to analyze user data and generate personalized recommendations. 

## Technologies Used
- **Python**: Main programming language used for the recommendation algorithms.
- **Flask**: Web framework used for the user interface and serving the recommendation system.
- **Scikit-learn**: Machine learning library for building the recommendation models.
- **Pandas**: Data manipulation and analysis library for handling datasets.
- **NumPy**: Essential library for numerical computations.
- **DVC**: Data Version Control system for managing large datasets and models.
- **Docker**: Used to containerize the application for easy deployment.
